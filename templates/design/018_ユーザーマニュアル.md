# ユーザーマニュアル / 管理者マニュアル

## 1. はじめに
### 1.1 本書の目的
本システム「〇〇システム」の操作方法、および管理手順について解説します。
### 1.2 推奨環境
* OS: Windows 10/11, macOS 最新版
* Browser: Google Chrome, Microsoft Edge 最新版

## 2. 一般ユーザー向け操作ガイド
### 2.1 ログインする
1. ログイン画面（URL）にアクセスします。
2. 管理者から配布されたIDとパスワードを入力します。
3. [ログイン]ボタンをクリックします。
   > [!NOTE]
   > 初回ログイン時はパスワードの変更が求められます。

### 2.2 〇〇データを登録する
1. メニューから[新規登録]を選択します。
2. 登録フォームに必要な情報を入力します。
   - **必須項目**には「*」が付いています。
3. [保存]ボタンをクリックします。
4. 「登録が完了しました」というメッセージを確認します。

### 2.3 困ったときは（FAQ）
* **Q: パスワードを忘れました**
    * A: ログイン画面の「パスワードをお忘れの方」リンクから再設定を行ってください。
* **Q: エラーが表示されます**
    * A: エラーメッセージを控えて、サポートデスクへ連絡してください。
